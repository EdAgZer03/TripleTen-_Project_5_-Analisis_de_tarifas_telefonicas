# 📶 Análisis Estadístico de Tarifas Telefónicas

**Resumen**  
Proyecto de análisis exploratorio para comparar planes telefónicos (Surf vs Ultimate) y detectar oportunidades de upsell.

**Objetivo**  
Analizar diferencias en consumo e ingresos por excedentes entre planes para proponer recomendaciones comerciales.

---

## 🔍 Qué hice
- Limpieza y consolidación de datos de ~500 usuarios → base reproducible para análisis.  
- Exploratory Data Analysis (EDA) → detección de outliers y patrones de consumo.  
- Pruebas de hipótesis (t‑test) → validación estadística de diferencias entre planes.  
- Visualizaciones comparativas → gráficos para producto y ventas.

---

## 🛠 Tecnologías
Python · Pandas · NumPy · SciPy · Matplotlib · Seaborn · Jupyter Notebook
